Functions Problem Set




Premise

The purpose of this exercise was to explore and assess my knowledge of Javascript arrays using the assignment given to me in this web developer course. You can access and use the problems using the console in your browser's developer tools.



Problems

printReverse(): A function that takes an array as an argument and prints out the array's elements in reverse order.

isUniform(): A function that takes an array as an argument and returns true if all of the elements in the array are identical.

sumofArray(): A function that accepts an array of numbers and returns the sum of all of the numbers in the array.

max(): A function that accepts an array of numbers and returns the maximum number in the array.
